TokenCrawler Tests

With these files you'll be able to see TokenCrawler in action.
Put the sample .txt files and the test script (test.bat) on the same directory as the application (where TokenCrawler.exe is) and in a command line execute "test.bat".
Check the results on screen or on the files that were produced.
